   # Design: 신임 팀장을 위한 익명 코호트 연속성 플랫폼

   ## Problem Statement

   기업에서 신임 팀장(파트장)으로 전환되는 시점(6-18개월)은 커리어에서 가장 취약한 구간이다.
   삼중고(관리 + 실무 + 보고)와 동료 관계의 구조적 변화로 심리적 고립이 발생하지만,
   직장 내 심리상담 채널은 "멘탈이 약하다"는 낙인 때문에 사용 불가능하다.
   현재 해결책: 음주, 주식 몰두, 친한 동료와의 커피 — 전부 비용이 크고 문제를 해결하지 못함.

   ## Demand Evidence

   - 삼성전자 18년차 과장 말년, 파트장 역할 수행 중. 자기보다 나이 많은 선배에게 하위고과를 줘야 하는 상황.
   - "하소연할 데가 없음" — 동료들은 관리직 이후 편하게 대해주지 않음.
   - 파트장 교육 현장에서 같은 고민을 가진 사람들이 자연스럽게 모이지만, 교육 종료 후 연결이 소멸.
   - "실제 관리직을 위한 마음건강센터는 존재하지 않음" — 사용자 직접 확인.
   - 실패의 결과: 18년 커리어 + 부장 진급 + 가장으로서의 역할 동시 위협.

   ## Status Quo

   | Workaround | 실제 비용 |
   |---|---|
   | 음주 | 건강, 가족 관계, 다음날 생산성 |
   | 주식 몰두 | 재무 리스크, 감정 회피 |
   | 친한 동료와 커피 | 단발성, 확장 불가, 동료 부담 |
   | 파트장 교육 (스킬 위주) | 감정 배출 없음, 교육 후 코호트 소멸 |

   경쟁자는 다른 앱이 아니다. **경쟁자는 현재의 음주 습관이다.**

   ## Target User & Narrowest Wedge

   **Target**: 삼성전자, LG, SK 등 대기업 신임 파트장/팀장. 연차 15-20년, 관리직 전환 후 3-18개월.

   **Narrowest Wedge — 파트장 교육 코호트 연속성**:
   - 기존 파트장 교육(삼성인력개발원, 외부 HRD 업체)이 끝난 뒤
   - 교육에서 형성된 코호트를 6개월간 익명으로 이어주는 플랫폼
   - 회사가 구매 → 개인 결제/낙인 없음
   - 주 1회 익명 체크인 + CBT 기반 감정 태깅 + 코호트 내 익명 공유

   ## Constraints

   1. 익명성이 절대 조건 — 기업 HR에 데이터 노출 불가
   2. "심리상담"이라는 단어를 사용하면 안 됨 — 프레이밍은 "리더십 동료 그룹" 또는 "팀장 동료 모임"
   3. 개인 B2C 결제 모델 불가 — 기업 계약(B2B) 필요
   4. 모바일 우선 — 업무 중 짬내서 쓸 수 있어야 함

   ## Premises

   1. 핵심 니즈는 "AI 조직관리 인사이트"가 아니라 "익명 동류 집단 + 안전한 배출구"다.
   2. 타겟 유저는 "심리상담 앱" 프레이밍에 거부감이 있다. 제품이 그렇게 보이면 안 된다.
   3. 파트장 교육 프로그램이 자연스러운 유통 채널(wedge)이다.
   4. 개인 B2C 앱보다 코호트 기반 익명 커뮤니티 모델이 이 문제에 더 적합하다.

   ## Approaches Considered

   ### Approach A: 익명 커뮤니티 MVP (Minimal Viable)
   - 카카오 오픈채팅 or 비공개 커뮤니티 (앱 없이 시작)
   - 파트장 교육 수료자에게 직접 연락해 커뮤니티 초대
   - Effort: S / Risk: Low
   - **용도**: 수요 검증 실험 (2주)

   ### Approach B: 코호트 기반 익명 앱 (Ideal Architecture)
   - 신임 팀장 5-8명 무작위 익명 코호트 매칭
   - CBT 기반 주간 체크인 + 코호트 내 익명 공유
   - AI: "당신만 그런 게 아님" 집계 데이터 제공
   - Effort: L / Risk: Med (닭-달걀 문제 있음)

   ### Approach C: 파트장 교육 기관 파트너십 (Recommended)
   - 파트장 교육 운영사에 "코호트 연속성 플랫폼"으로 납품
   - 회사가 구매 → 낙인 없음, 코호트 신뢰 기제작
   - B2B 단가 높음, 유통 채널 기제작
   - Effort: M / Risk: Med (파트너십 협상 필요)
